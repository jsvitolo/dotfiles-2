## Install

```
git clone git@github.com:leandrocp/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
bash install.sh
```

## MacOS

```
cd ~/.dotfiles
bash osx.sh
bash apps.sh
```

## Dependencies
* zsh
* https://github.com/zsh-users/antigen
* https://github.com/tmux/tmux
* https://github.com/asdf-vm/asdf
* https://github.com/dracula/dracula-theme
